# قالب موقع عربي (RTL)

هذا القالب جاهز للتعديل والنشر مجانًا على GitHub Pages.

## طريقة النشر (GitHub Pages)
1) سجّل دخولك في GitHub وأنشئ مستودعًا جديدًا باسم `my-website` (أو أي اسم).
2) ارفع هذه الملفات كاملة إلى المستودع.
3) من تبويب **Settings** > قسم **Pages**:
   - اختر **Deploy from a branch**
   - **Branch:** main  
   - **Folder:** / (root)  
   - احفظ.
4) بعد دقيقة تقريبًا سيظهر رابط موقعك مثل:  
   `https://USERNAME.github.io/my-website/`

> تريد دومين خاص؟ من نفس صفحة **Pages** أضف اسم الدومين، ثم حدّث سجلات الـ DNS من مزوّد النطاق إلى GitHub Pages.

## تخصيص سريع
- عدّل العناوين والنصوص داخل `index.html`.
- ضع صور أعمالك داخل مجلد `assets` وغيّر المسارات.
- لتفعيل نموذج التواصل، عدّل الدالة `handleSubmit` في `script.js` واربطها بخدمة مثل Formspree.

## ملاحظات
- لا يعتمد القالب على مكتبات خارجية؛ سريع وخفيف.
- الاتجاه `dir="rtl"` وخطوط مناسبة للعربية.
